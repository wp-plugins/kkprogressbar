== Description ==

Plugin shows/indicates progress that has been made on projects or articles.

= Shortcode =

Tag can be inserted in a random place on web pages or in articles.

Tag has to hold one attribute:
* "IDKKPB" is a mandatory one. The ID of progressbar needs to be provided (example: `idkkpb = "3"`)

Correct tag should be as following:
`[kkprogressbar idkkpb="3"]`


DEMO: http://krzysztof-furtak.pl/2010/06/wp-kk-progressbar-plugin/
Author Site: http://krzysztof-furtak.pl/

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
